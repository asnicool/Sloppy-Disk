# Playlist View Enhancement Plan

## Overview
Add 3 features to the playlist (QueueView):
1. Collapsed mode as default, persisted across sessions
2. Drag-to-trashcan for removing albums/tracks
3. "Remove played albums" button

---

## Feature 1: Collapsed by Default + Persistence

### Changes Required

**File: `frontend/src/views/QueueAlbumStrip.vue`**

- Change `isCompact` default from `false` to `true`
- Add localStorage persistence:
  - On mount: Load from localStorage (or default to `true`)
  - On toggle: Save to localStorage

---

## Feature 2: Drag-to-Trashcan

### UI Design
- A trashcan icon that appears at the bottom of the playlist area when dragging starts
- The trashcan should be prominent (red accent, large icon)
- Visual feedback: highlight trashcan when draggable hovers over it
- Drop on trashcan triggers removal

### Implementation

**File: `frontend/src/views/QueueAlbumStrip.vue`**
- Add `isDragging` ref to track drag state
- Add trashcan drop zone below the album list
- Show trashcan when any drag operation starts
- Handle drop event to remove the dragged item

**Drag behavior:**
- When dragging an album (via album handle): drop on trashcan removes entire album
- When dragging a track (via track handle): drop on trashcan removes just that track

---

## Feature 3: Remove Played Albums Button

### UI Design
- A button always visible in the header area (next to the collapse/expand toggle)
- Icon: trashcan with "sweep" or "clear played" concept
- Text: "Clear Played" or "Remove Played"
- Color: Subtle by default, maybe red accent on hover
- Shows count of albums that will be removed

### Logic
- An album is "fully played" if ALL its tracks have `isPlayed = true`
- If the same album appears multiple times (split), each part is treated independently
- If an album has even ONE unplayed track, it is kept
- Currently playing album is always kept (even if some tracks are played)

### Implementation

**File: `frontend/src/views/QueueAlbumStrip.vue`**
- Add computed `playedAlbums` that returns groups where all tracks are played
- Add function `removePlayedAlbums()` that iterates and removes all played groups
- Add button in header with count badge

## Acceptance Criteria

1. **Collapsed default**: When opening playlist for the first time, albums are shown in collapsed (compact) mode
2. **Persistence**: Collapse state survives page refresh and navigation away/back
3. **Trashcan**: When dragging any album or track, a trashcan appears; dropping removes the item
4. **Remove played**: Button removes all fully-played albums; keeps albums with unplayed tracks; split album parts are handled independently
